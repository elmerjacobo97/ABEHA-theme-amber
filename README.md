## üåì Theme Variants

### ABEHA Amber Dark

Perfect for low-light environments with a pure dark background (#1a1a1a) and amber highlights.

### ABEHA Amber Light

Ideal for well-lit spaces with a clean white background (#ffffff) and amber accents.

## üì¶ Installation

1. Open VS Code
2. Go to Extensions (`Cmd+Shift+X` on macOS or `Ctrl+Shift+X` on Windows/Linux)
3. Search for "ABEHA Amber Theme"
4. Click Install
5. Press `Cmd+K Cmd+T` (or `Ctrl+K Ctrl+T`) to open the theme selector
6. Choose **ABEHA Amber Dark** or **ABEHA Amber Light**
